# Band With Me — Design Explorations Round 4

Round 4 reagiert auf das Feedback:
- weniger flach
- weniger generisch / KI-basic
- weniger nur große dunkle Karten
- kleinere, normalere Buttons
- weniger übertriebenes Padding
- modernere App-/Web-Patterns
- mehr Variation zwischen den Seiten

## Typografie
- **Headlines:** `Space Grotesk`
- **Body/UI:** `Inter`

Warum?
- `Space Grotesk` gibt den Überschriften mehr Eigencharakter als reines Inter
- `Inter` hält Controls, Listen und Produkttext modern und gut lesbar

## Struktur
Für jede Seite gibt es **zwei Varianten**:
- eine emotionalere / markantere
- eine ruhigere / nativer wirkende

## Design-Checklisten-Ziel
Diese Runde versucht bewusst stärker zu verbessern bei:
- Contrast
- Hierarchy
- White Space
- Unity
- weniger card-heavy repetition
