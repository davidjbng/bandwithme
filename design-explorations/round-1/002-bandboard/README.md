## Variant: Bandboard

### Design stance
Eine produktivere Richtung mit Dashboard-Gefühl, stärkerer Informationsdichte und schneller Scanbarkeit.

### Key choices
- Schwerpunkt: Onboarding + Termine + Songs in einem ersten Gesamtbild
- Signalwirkung: klare Terminzustände, weiche Card-Struktur, bandtaugliche CTA-Hierarchie
- Ziel: frühe Richtungsentscheidung vor Produktions-UI

### Best for
- eine erste UI-Entscheidung über Stil, Dichte und Tonalität
