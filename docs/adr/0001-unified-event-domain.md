# Vereinheitlichte Termin-Domäne für Proben und Auftritte

Band With Me behandelt Probe und Auftritt als zwei Typen derselben Termin-Domäne statt als getrennte Systeme. Das ist schwer später zurückzudrehen, wäre ohne Erklärung überraschend und ist das Ergebnis eines echten Trade-offs: getrennte Bereiche wirken im UI klar, würden aber Datenmodell, Kalenderlogik, Zusagen und Listen unnötig doppeln; deshalb teilen beide eine gemeinsame Terminbasis mit typspezifischen Erweiterungen.
