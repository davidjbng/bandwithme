# Song-Status entsteht aus Kontext statt aus Kategorien

In V1 bekommen Songs keine festen Kategorien wie „neu“, „in Arbeit“ oder „auftrittsreif“. Stattdessen ergibt sich ihr aktueller Status aus den Kontexten Song-Voting, Probefokus und Setlist; das hält das Modell leichter, vermeidet doppelte Pflege und bleibt später erweiterbar, obwohl eine klassische Kategorie-Lösung auf den ersten Blick naheliegend wäre.
