# Probenserien sind eigenständige Objekte

Band With Me behandelt eine Probenserie nicht nur als Erstellhilfe für kopierte Einzeltermine, sondern als eigenständiges Domänenobjekt. Das ist schwer später zurückzudrehen, wäre ohne Kontext überraschend und ist Ergebnis eines echten Trade-offs: eine reine Erzeugungsfunktion wäre in V1 einfacher, würde aber spätere Änderungen, Pausen, Serienpflege und Kalenderlogik unnötig erschweren.
